Description

RubyScope is a GUI for ruby-debug and a Google Chrome app. After installing it, you can connect to a remote ruby-debug running by specifying a host and port.

Features

RubyScope comes with most of the things you're used to in a full-fleged graphical debugger including:

  • Show Stack (and jump-to-frame)
  • Toggle and List Breakpoints
  • List Open Files and File Contents
  • Syntax-highlighted Code Pane
  • Interactive IRB-like Console

How It Works

RubyScope connects directly to your remote debugger using an experimental TCP socket feature available to Chrome apps. No whacky gems are required, no special IDEs are required either, and nothing at all requires Java.
