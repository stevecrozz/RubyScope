)
jQuery.Deferred

Defined in lib/debugger-ui.js:67

Ask the adapter for data. Provide a dataPoints object to collect all the data provided by the adapter.

If you don't provide any deferreds, provideData will create one for each dataProvider internally. If you do provide deferreds, provideData will not request data from that dataProvider and will assume the data is available as soon as the given deferred is resolved.

Parameters:

  • dataPointNames Array

    list of named data requirements

  • dataPoints Object

    data collection object

  • deferreds Object

    keyed by data provider name

Returns:

jQuery.Deferred: resolved when data requirements are fulfilled
